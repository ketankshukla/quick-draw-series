# HeyGen Video Trailer Prompt Template

Use this template to create book trailer videos with HeyGen AI video generation.

---

## Video Specifications

- **Duration:** 60-90 seconds
- **Aspect Ratio:** 16:9 (landscape) for YouTube, 9:16 (portrait) for TikTok/Reels
- **Resolution:** 1080p minimum
- **Style:** Cinematic, dramatic, professional

---

### Scene 4: Resolution Tease (45-60 seconds)

### Scene 5: Call to Action (60-75 seconds)

## Background Music Suggestions

- Cinematic orchestral (epic fantasy)
- Dark ambient electronic (thriller/philosophical)
- Inspiring piano and strings (self-help/educational)
- Tension-building percussion (action/adventure)

---

## Post-Production Notes

- Add subtle letterboxing (black bars top/bottom) for cinematic feel
- Use consistent color grading matching the book's accent color
- Include Amazon link in video description
- Add end screen with clickable links (YouTube)
